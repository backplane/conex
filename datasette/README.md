# datasette

[uv:alpine](https://github.com/astral-sh/uv/pkgs/container/uv)-based image for [datasette](https://docs.datasette.io/en/stable/), the "open source multi-tool for exploring and publishing data"

As [the site](https://datasette.io/) says:

> Datasette is aimed at data journalists, museum curators, archivists, local governments, scientists, researchers and anyone else who has data that they wish to share with the world. It is part of a wider ecosystem of 44 tools and 154 plugins dedicated to making working with structured data as productive as possible.

This image is built from the latest [datasette release on pypi](https://pypi.org/project/datasette/#history).
